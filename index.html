Autonomous Discovery Methods

How it works

Domain Discovery

Certificate Transparency logs, DNS enrichment, and subdomain expansion uncover new scope daily.

Repository Scanning

Tracks trending and recently updated repositories for SAST and dependency-risk analysis.

Web Crawling

Explores URLs, forms, and APIs to map application surfaces and queue targeted scanner workflows.

Blockchain Monitoring

Monitors smart-contract deployments, transaction behavior, and risky contract patterns in real time.

API Discovery

Discovers public API directories, OpenAPI endpoints, and GraphQL interfaces for coverage expansion.

Contact & Notify

Uses security.txt, WHOIS, and repo contacts to execute structured responsible disclosure outreach.
